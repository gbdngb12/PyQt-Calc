# PyQt-Calc
QT Based Calculator


![image](https://user-images.githubusercontent.com/104804087/205068189-d8fe8010-4992-47a6-9bfe-d7d4c91461a5.png)

## Install
Requires pip and PyQt5
```bash
sudo apt install python3-pip
```
```bash
pip install PyQt5
```
